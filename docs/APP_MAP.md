# Карта приложения v2

## 1. До входа

```text
Приложение
└── Добро пожаловать
    ├── Продолжить с Google
    ├── Регистрация по email
    │   └── Подтверждение email
    ├── Вход по email
    │   └── Восстановление пароля
    └── После авторизации
        └── Выбор профилей
            ├── Игрок — обязателен
            ├── Тренер — дополнительно
            ├── Компания / клуб — дополнительно
            ├── Заполнение выбранных профилей
            ├── Разрешения
            └── Главная
```

## 2. Глобальная оболочка

```text
Верхняя панель
├── Аватар активного профиля
│   └── Переключатель
│       ├── Игрок
│       ├── Тренер
│       ├── Организация
│       ├── Добавить профиль
│       └── Управление профилями

Нижняя навигация
Главная · События · Чаты · Клубы · Профиль
```

## 4. События

```text
События
├── Все — публичный каталог
├── Участвую — участия, заявки и приглашения
├── Управляю — созданные события и задачи
│   └── Управление конкретной сущностью
├── Мои ближайшие — закреплённый блок
│   └── Канонический экран сущности
├── Черновики
│   └── Продолжить создание
├── Категории
│   ├── Все
│   ├── Игры
│   ├── Тренировки
│   ├── Турниры
│   └── Сезоны
├── Поиск
├── Фильтры
├── Карта
└── Создать
```

Карточка одного события меняет действие:

```text
Посторонний → Присоединиться / Подать заявку
Участник → Открыть / Оплатить / Чат
Организатор → Управлять
Черновик → Продолжить
```

## 14. Главное правило связей

Одна сущность имеет один публичный экран и один кабинет управления. Разные разделы дают разные входы, но не создают копии:

```text
Главная ─┐
События ─┤
Календарь ├→ Канонический экран сущности → Управление при наличии прав
Чаты ────┤
Профиль ─┘
```
