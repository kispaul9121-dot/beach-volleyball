# Карта приложения

## Нижняя навигация

```text
Главная · Играть · Чаты · Клубы · Профиль
```

## Главная

```text
Главная
├── Ближайшее событие → канонический экран события
├── Сегодня → Мой календарь / выбранный день
├── Приглашения → детали приглашения
├── Требуют внимания
│   ├── Оплата
│   ├── Подтверждение участия
│   └── Ввод результата
├── Рекомендации → детали сущности
└── Быстрые действия
    ├── Найти игру → Играть / Игры
    ├── Создать игру → Создание игры
    ├── Найти клуб → Клубы
    └── Открыть календарь → Профиль / Мой календарь
```

## Играть

```text
Играть
├── Все
├── Игры

├── Турниры
├── Сезоны
```
