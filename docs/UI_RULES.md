# UI-правила

## Нижняя навигация

Ровно пять пунктов: Главная, Играть, Чаты, Клубы, Профиль.

## Размер и плотность

- Не более трёх быстрых фильтров одновременно.
- Остальные фильтры открываются в bottom sheet.
- На карточке одно главное действие.
- Кликабельной является вся карточка, а не только маленькая стрелка.
- Минимальная зона нажатия — 44×44 pt / 48×48 dp.
- Детали не помещаются внутрь перегруженной карточки; они открываются отдельным экраном.

## Поиск

- В «Играть» поиск по играм, тренировкам, турнирам и сезонам.
- В «Клубах» поиск по клубам, кортам, тренерам и турам.
- В «Чатах» поиск по разговорам и участникам.
- В «Профиле → Мои игры» нет публичного поиска города.

## Карточки

Общая основа:

- тип сущности;
- название;
- дата и время;
- место;
- организатор;
- цена или бесплатно;
- статус;
- одно главное действие.

## Возврат

- Корневые вкладки не показывают кнопку назад.
- Вложенные экраны показывают назад.
- Мастера создания используют «Отмена» и «Сохранить черновик».
- Deep link всегда имеет `back_fallback`.
