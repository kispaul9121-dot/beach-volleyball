# Пути пользователей

## Игрок ищет и вступает в игру

```text
Играть → Игры → Поиск/фильтры → Детали игры → Присоединиться
→ Оплата или подтверждение → Профиль → Мои игры → Участвую
→ Чат игры → Игра → Результат → Завершённые
```

## Игрок создаёт одноразовую игру из профиля

```text
Профиль → Мои игры → Созданные → Создать игру
→ Мастер создания → Черновик или публикация
→ Мои игры / Созданные → Управлять
→ Участники → Корты и раунды → Результаты → Завершить
```

## Игрок создаёт игру из публичного каталога

```text
Играть → Создать → Игра → тот же мастер /games/create
→ публикация → детали игры
```

Важно: это не второй мастер. Меняется только `returnTo`.

## Игрок участвует в сезоне ротации пяти игроков

```text
Играть → Сезоны → Детали сезона → Вступить
→ Игровые дни → Регистрация на ближайший день
→ конкретный игровой день → назначенный корт и раунды
→ результаты → общая таблица сезона
```

## Организатор сезона

```text
Профиль → Мои турниры и сезоны → Созданные
→ Сезон → Управлять → Создать игровой день
→ выбрать 2 или 3 корта → вместимость 10 или 15 игроков
→ подтвердить участников → сгенерировать ротацию
→ опубликовать составы → внести результаты → обновить таблицу
```

## Тренер

```text
Профиль → Мои тренировки → Создать
→ программа, уровень, места, корт, цена
→ публикация → набор участников → чат группы
→ посещаемость → завершение
```

## Клуб

```text
Кабинет клуба → Создать активность
→ игра / тренировка / турнир / сезон / тур
→ выбрать собственный корт и доступность
→ оплата → публикация → управление участниками
```

## Компания или клуб создаёт тур

```text
Кабинет организатора → Туры → Создать
→ даты, место, программа, проживание, цена
→ публикация в Клубы / Туры
→ бронирования → оплаты → чат → проведение
```
