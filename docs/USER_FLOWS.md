# Пути пользователей

## 1. Регистрация с несколькими профилями

```text
Добро пожаловать
→ Google или email
→ подтверждение аккаунта
→ выбрать Игрок + Тренер + Компания
→ заполнить игрока
→ заполнить или отложить тренера
→ заполнить или отложить организацию
→ разрешения
→ Главная игрока
→ переключатель actor
→ Главная тренера / организации
```

## 2. Игрок ищет и вступает в игру

```text
Играть
→ Игры
→ поиск / фильтры
→ Детали игры
→ Присоединиться / Подать заявку
→ Оплата или подтверждение
→ игра появляется в Главной, Календаре и Мои игры
→ Чат
→ Раунды / Матчи
→ Результат
→ Завершённые
```

## 3. Игрок создаёт игру

```text
Главная или Играть или Профиль / Мои игры
→ Создать игру
→ Создать как: Игрок
→ формат, место, корты, участие, цена
→ черновик или публикация
→ игра появляется в Управляю
→ Управление игрой
→ участники
→ корты и раунды
→ результаты
→ завершение
```

## 4. Организатор быстро возвращается в игру

```text
Главная → Управляю → Управление игры
или
Играть → Управляю → Управление игры
или
Профиль → Мои игры → Созданные → Управление
```

Все пути ведут на один `/games/:gameId/manage`.

## 5. Смена создателя в черновике

```text
Создание игры как Игрок
→ открыть Создать как
→ выбрать Организацию
→ подтвердить смену
→ повторная проверка полей и финансов
→ выбрать корт организации
→ публикация от имени организации
```

## 6. Игрок создаёт совместную практику

```text
Создать тренировку
→ Создать как: Игрок
→ режим Совместная практика
→ дата, корт, уровень, разделение расходов
→ публикация
```

Публичный экран не называет игрока профессиональным тренером.

## 7. Тренер создаёт занятие и группу

```text
Переключиться на Тренера
→ Профиль / Мои тренировки
→ Создать группу
→ участники и расписание
→ Создать тренировку из группы
→ программа, уровень, места, цена
→ публикация
→ заявки и оплаты
→ посещаемость
→ статистика тренера и группы
```

## 8. Организация создаёт тренировку

```text
Переключиться на Организацию
→ Мои тренировки
→ Создать
→ назначить тренера-сотрудника
→ выбрать свой корт
→ цена / абонемент
→ публикация
→ сотрудник ведёт посещаемость по своей роли
```

## 9. Игрок участвует в турнире

```text
Играть → Турниры
→ Детали турнира
→ регистрация
→ оплата
→ Мои турниры / Участвую
→ жеребьёвка опубликована
→ Мой следующий матч
→ Матч
→ подтвердить результат
→ таблица / сетка / места
```

## 10. Создание турнира

```text
Мои турниры и сезоны
→ Создать турнир
→ Создать как
→ участники
→ формат
→ правила и тай-брейки
→ регистрация
→ корты и расписание
→ публикация
→ Управление турниром
```

## 11. Швейцарский турнир

```text
Закрыть регистрацию
→ подтвердить участников
→ посев
→ создать черновые пары раунда 1
→ проверить и опубликовать
→ матчи и результаты
→ завершить раунд
→ создать пары следующего раунда по score groups
→ повторять
→ финальная таблица
```

## 12. Полное распределение 32

```text
32 участника
→ посев
→ раунд 1: 16 матчей
→ победители и проигравшие расходятся по placement-веткам
→ ещё 4 раунда по 16 матчей
→ 80 завершённых матчей
→ уникальные места 1–32
```

## 13. Сезон ротации пяти

### Участник

```text
Играть → Сезоны → Детали
→ Вступить
→ Игровые дни
→ регистрация на день
→ Игровой день
→ корт, партнёры, соперники, отдых
→ результаты
→ сезонная таблица
```

### Организатор

```text
Мои турниры и сезоны → Организую
→ Управление сезоном
→ создать игровой день
→ выбрать 2 или 3 корта
→ вместимость 10 или 15
→ подтвердить участников
→ сгенерировать группы и раунды
→ проверить инварианты
→ опубликовать
→ результаты
→ пересчитать таблицу
```

## 14. Игрок создаёт неформальную поездку

```text
Мои туры → Создать
→ Создать как: Игрок
→ Неформальная поездка
→ даты, план, бюджет, участники
→ чат группы
→ совместная организация
```

## 15. Тренер создаёт лагерь

```text
Переключиться на Тренера
→ Создать тур
→ Тренировочный лагерь
→ программа, уровень, тренеры, проживание
→ цена
→ публикация
→ заявки, оплаты, чат
→ проведение
```

## 16. Организация создаёт коммерческий тур

```text
Переключиться на Организацию
→ Создать тур
→ Коммерческий пакет
→ тарифы, размещение, трансфер, документы
→ сотрудники и тренеры
→ платежный график
→ публикация в Клубы / Туры
→ бронирования
→ оплаты и возвраты
→ проведение
```

## 17. Календарь всех профилей

```text
Профиль → Мой календарь
→ Все профили
→ увидеть личную игру, тренировку тренера и событие организации
→ фильтр Участвую / Управляю
→ открыть канонический экран или manage
```

## 18. Чат открывает контекст

```text
Чаты → чат сущности
→ системное сообщение: опубликован раунд
→ Открыть событие
→ канонический турнир
→ Мой следующий матч
```

## 19. Потеря права сотрудника

```text
Сотрудник открыт в manage
→ владелец отзывает membership
→ следующая серверная операция отклоняется
→ интерфейс обновляет права
→ предлагает публичный экран
```

## 20. Возврат и deep link

Любой прямой deep link открывает канонический экран. При отсутствии истории кнопка назад использует `back_fallback`. Manage deep link без разрешения не раскрывает внутренние данные.
