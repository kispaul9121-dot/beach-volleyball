# Клубы

- Screen ID: `clubs.main`
- Route: `/clubs`
- Variants: `player`, `trainer`, `organization`

## Категории

- Клубы;
- Площадки и корты;
- Тренеры;
- Туры.

Поиск, быстрые фильтры, расширенные фильтры и карта относятся к выбранной категории.

## Игрок

Приоритеты: ближайшие площадки, клубы, подходящие тренеры и публичные туры.

## Тренер

Дополнительно:

- свой публичный профиль;
- клубы, где тренер работает;
- доступные площадки;
- собственные тренерские туры.

## Организация

Дополнительно сверху:

- `Моя организация`;
- собственные площадки;
- сотрудники / тренеры;
- кнопка управления клубом.

Публичный каталог остаётся ниже и не исчезает.

## Карточки

### Клуб

Название, город, площадки, ближайшие события, тренеры, рейтинг.

### Площадка

Адрес, корты, покрытие, доступность, ближайший слот.

### Тренер

Имя, проверка, специализации, ближайшая тренировка.

### Тур

Тип создателя, даты, направление, цена / бюджет, свободные места.
