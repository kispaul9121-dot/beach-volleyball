# Играть

- Screen ID: `play.main`
- Route: `/play`
- Variants: `player`, `trainer`, `organization`

Раздел сочетает публичный каталог и быстрый доступ к актуальным сущностям активного профиля. Полная история остаётся в Профиле.

## Верхняя панель

- активный профиль → actor switcher;
- поиск;
- фильтры;
- список / карта;
- кнопка `Создать` с доступными типами сущностей.

## Верхние персональные блоки

Перед публичной выдачей показываются компактные горизонтальные блоки:

1. `Управляю` — опубликованные сущности активного профиля с ближайшими действиями;
2. `Мои ближайшие` — подтверждённые участия;
3. `Черновики` — только если есть незавершённые создания.

Блоки можно свернуть, но нельзя полностью прятать управляемые сущности в Профиле.

## Категории

- Все;
- Игры;
- Тренировки;
- Турниры;
- Сезоны.

Туры остаются в «Клубы → Туры», но связанные тренерские лагеря могут быть рекомендованы в соответствующем контексте.

## Карточка публичной сущности

Показывает:

- тип и формат;
- дату и место;
- уровень;
- организатора и тип профиля;
- цену;
- свободные места;
- статус пользователя.

Дополнительные плашки:

- `Вы организатор`;
- `Вы участвуете`;
- `Требуется оплата`;
- `Заявка отправлена`;
- `Лист ожидания`.

## Поведение кнопки

- посторонний → `Присоединиться` / `Подать заявку`;
- участник → `Открыть`;
- организатор → `Управлять`;
- черновик → `Продолжить`.

## Фильтры

Дата, расстояние, формат, уровень, цена, свободные места, тип организатора, только мои, только управляемые.

## Состояния

- loading skeleton;
- нет результатов с предложением сбросить фильтры;
- нет управляемых сущностей;
- геолокация недоступна;
- offline;
- частичная ошибка карты.
