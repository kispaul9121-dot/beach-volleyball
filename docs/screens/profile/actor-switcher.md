# Переключатель и управление профилями

## actor.switcher

Открывается нажатием на аватар в общей верхней панели.

## Содержимое bottom sheet

- текущий активный профиль;
- список доступных профилей с типом, аватаром и статусом;
- незавершённые профили с прогрессом;
- `Добавить профиль`;
- `Управление профилями`.

## Нажатие на профиль

1. Проверить, доступен ли actor пользователю.
2. Если открыт чистый экран — переключить сразу.
3. Если открыт мастер создания с несохранёнными изменениями — спросить:
   - продолжить создание от текущего профиля;
   - сменить создателя;
   - отменить переключение.
4. Обновить `active_actor_id`.
5. Пересчитать разрешения и обновить текущий экран.

## profile.actor_profiles — `/profile/actors`

Карточка каждого профиля:

- имя / название;
- тип;
- статус проверки;
- прогресс заполнения;
- публичная видимость;
- выплаты;
- кнопка редактирования.

Действия:

- добавить тренера;
- добавить организацию;
- завершить заполнение;
- открыть публичный профиль;
- подключить выплаты;
- покинуть организацию;
- деактивировать профиль.

## Ограничения

- обязательный player-профиль нельзя удалить отдельно от аккаунта;
- actor с активными платными сущностями нельзя удалить без передачи или завершения;
- владелец организации не может покинуть её без назначения нового владельца;
- удаление профиля не удаляет финансовую и аудиторскую историю.

## Состояния

- только один профиль;
- несколько профилей;
- приглашение в организацию;
- проверка тренера;
- отклонённая проверка;
- потеря доступа;
- offline.
