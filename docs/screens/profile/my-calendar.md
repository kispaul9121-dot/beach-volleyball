# Мой календарь

- Screen ID: `profile.calendar`
- Route: `/profile/calendar`
- Parent: `profile.main`

## Назначение

Спортивный календарь всех подтверждённых и ожидающих действий пользователя.

## Представления

- День
- Неделя
- Месяц

## Типы элементов

- одноразовая игра;
- тренировка;
- турнир;
- игровой день сезона;
- тур;
- бронирование корта;
- дедлайн оплаты или регистрации.

## Взаимодействия

- тап по карточке → канонический экран сущности;
- свайп по датам;
- фильтр по типам;
- кнопка «Сегодня»;
- индикаторы статусов: подтверждено, ожидает оплаты, приглашение, отменено;
- длинное нажатие или меню → добавить во внешний календарь, скрыть, отменить участие, открыть чат.

## Возврат

Назад → Профиль. При deep link fallback `/profile`.
