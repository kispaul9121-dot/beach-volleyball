# Мои игры

- Screen ID: `profile.my_games`
- Route: `/profile/games`
- Parent: `profile.main`

## Назначение

Полный личный архив одноразовых игр: участия человека и игры, созданные или управляемые активным actor-профилем. Не является публичным каталогом.

## Шапка

- назад в Профиль;
- название `Мои игры`;
- фильтр actor: `Все профили` / активный профиль;
- `+` → создание игры с текущим actorId.

## Главные вкладки

### Участвую

Фильтры:

- Предстоящие;
- Требуют действий;
- Завершённые;
- Отменённые.

Карточка показывает:

- дату, место и формат;
- организатора;
- статус участия;
- оплату;
- чат и непрочитанные сообщения;
- результат после завершения.

Главные действия:

- Открыть;
- Оплатить;
- Открыть чат;
- Подтвердить изменение;
- Отменить участие через меню.

### Созданные / Управляю

Фильтры:

- Опубликованные;
- Требуют действий;
- Черновики;
- Завершённые;
- Отменённые.

Карточка показывает:

- actor-профиль, от имени которого создана игра;
- участников X из Y;
- заявки и лист ожидания;
- неоплаченные регистрации;
- опубликованы ли раунды;
- незаполненные результаты;
- ближайшее управленческое действие.

Главные действия:

- Управлять;
- Продолжить создание;
- Открыть результаты;
- Дублировать;

## Быстрый доступ вне Профиля

Игры из вкладки `Созданные / Управляю` также показываются:

- на Главной в блоке `Управляю`;
- в верхней части раздела `Играть`;
- в календаре;
- в уведомлениях.

Профиль нужен для полного списка, а не для ежедневного поиска своих игр.

## Пустые состояния

- нет участий → кнопка `Найти игру`;
- нет созданных → кнопка `Создать игру`;
- нет игр активного actor → предложить `Все профили`;
- нет результатов фильтра → сбросить фильтры.

## Ошибки и права

- actor потерял право управления;
- игра перенесена другому владельцу;
- черновик создан другим actor-профилем;
- удалённая или отменённая игра;
- offline с последним состоянием.
