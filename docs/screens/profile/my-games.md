# Мои игры

- Screen ID: `profile.my_games`
- Route: `/profile/games`
- Parent: `profile.main`

## Назначение

Личный центр одноразовых игр. Не является публичным каталогом.

## Верхний уровень

- вкладка `Участвую`;
- вкладка `Созданные`;
- кнопка `+` → единый мастер `/games/create?returnTo=/profile/games?tab=created`.

## Участвую

Фильтры: Предстоящие, Требуют действий, Завершённые.

Карточка может показывать:

- подтверждённое участие;
- ожидание подтверждения;
- оплату;
- изменение места или времени;
- результат.

Главные действия: Открыть, Оплатить, Открыть чат.

## Созданные

Фильтры: Опубликованные, Черновики, Завершённые.

Карточка показывает управленческий статус:

- участников X из Y;
- заявки;
- задолженности;
- необходимость распределить корты;
- необходимость внести результат.

Главные действия: Управлять, Продолжить создание, Открыть результаты.

## Не входит

- игры города;
- рекомендации;
- фильтр «рядом»;
- тренировки, турниры и сезоны.
