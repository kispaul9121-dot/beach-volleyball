# Мои тренировки

- Screen ID: `profile.trainings`
- Route: `/profile/trainings`

## Назначение

Раздел объединяет записи игрока и управляемые занятия активного профиля тренера или организации.

## Верхний уровень

- actor-фильтр;
- вкладки `Записан` и `Провожу / Организую`;
- для тренера и организации — `Группы`;
- кнопка `+ Создать тренировку`, доступная тренеру и организации.

## Игрок: «Записан»

Фильтры:

- Предстоящие;
- Требуют оплаты;
- Завершённые;
- Отменённые.

Карточка:

- тренер и организация;
- программа и уровень;
- дата, место;
- статус записи;
- оплата;
- чат;
- отметка посещения после занятия.

Действия: Открыть, Оплатить, Открыть чат, Отменить запись.

## Тренер / организация: «Провожу»

Фильтры:

- Сегодня;
- Требуют действий;
- Предстоящие;
- Черновики;
- Завершённые.

Карточка:

- участники X из Y;
- заявки;
- неоплаченные места;
- программа заполнена / не заполнена;
- посещаемость не внесена;
- связанная группа;
- назначенный тренер.

Действия: Управлять, Посещаемость, Сообщить группе, Дублировать.

## Группы

Карточка группы показывает:

- расписание;
- участников;
- абонементы;
- следующий урок;
- задолженности;
- среднюю посещаемость.

Внутри группы: участники, расписание, абонементы, прогресс, сообщения, настройки.

## Пустые состояния

- игрок не записан → `Найти тренировку`;
- тренер ничего не проводит → `Создать тренировку`;
- нет групп → `Создать группу`;
- активный actor не может создавать тренировки → предложить добавить профиль тренера.

## Быстрый доступ

Сегодняшние занятия и незаполненная посещаемость показываются на Главной. Управляемые тренировки также закрепляются в разделе Играть.
