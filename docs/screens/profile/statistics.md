# Статистика

- Screen ID: `profile.statistics`
- Route: `/profile/statistics`
- Variants: `player`, `trainer`, `organization`

## Общие элементы

- активный профиль;
- период;
- фильтр типа сущности;
- карточки ключевых показателей;
- график динамики;
- список событий-источников;
- методика расчёта.

## Игрок

Показатели MVP:

- игры и матчи;
- победы / поражения;
- очки и разница;
- посещаемость;
- партнёры и совместные результаты;
- динамика рейтинга;
- результаты турниров и сезонов.

Ротация пяти игроков показывает личные очки, победы, разницу и места по игровым дням.

## Тренер

Показатели MVP:

- проведённые занятия;
- уникальные ученики;
- средняя заполняемость;
- посещаемость;
- повторные записи;
- отмены;
- выручка при подключённых выплатах;
- рейтинг и отзывы.

## Организация

Показатели MVP:

- количество событий;
- участники;
- заполняемость;
- загрузка кортов;
- выручка, возвраты и задолженности;
- эффективность типов событий;
- активность сотрудников.

## Правила расчёта

- статистика строится из событий результатов, посещаемости и платежей;
- итоговые агрегаты нельзя редактировать вручную;
- исправление исходного результата пересчитывает зависимые показатели;
- методика и дата последнего обновления видимы пользователю;
- приватные данные доступны только владельцу или сотруднику с разрешением.

## Состояния

- недостаточно данных;
- расчёт обновляется;
- часть источников недоступна;
- период без активности;
- скрытая статистика другого пользователя.
