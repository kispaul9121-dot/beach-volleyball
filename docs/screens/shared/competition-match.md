# Матч соревнования

- Screen ID: `competition.match`
- Route: `/matches/:matchId`

## Блоки

- турнир или сезон;
- раунд / группа / стадия;
- участники или команды;
- корт и время;
- статус;
- счёт по сетам;
- главный результат;
- судья / счётчик;
- история изменений;
- связанные сообщения.

## Состояния

- scheduled;
- called_to_court;
- in_progress;
- result_pending_confirmation;
- completed;
- disputed;
- walkover;
- cancelled.

## Участник

Видит время, корт, соперника, уведомление о вызове, результат и возможность подтвердить или оспорить результат по правилам турнира.

## Редактор результата

Доступен организатору, судье или счётчику:

- счёт сетов;
- очки;
- технический результат;
- причина;
- подтверждение.

Невалидный счёт блокируется. После подтверждения создаётся immutable result event и пересчитываются сетка / таблица.

## Исправление

Исправление завершённого результата требует:

- разрешения;
- причины;
- записи предыдущего значения;
- перерасчёта зависимых матчей и таблиц;
- предупреждения, если следующий раунд уже опубликован.

## Навигация

- открыть турнир / сезон;
- открыть раунд;
- открыть чат;
- для организатора — перейти в управление соревнованием.
