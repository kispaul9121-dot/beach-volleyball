# Детали игры

- Screen ID: `game.details`
- Route: `/games/:gameId`
- Variants: `guest`, `participant`, `organizer`

Канонический экран одноразовой игры. Один маршрут используется из Главной, Играть, Календаря, Чатов, уведомлений и Профиля.

## Общие блоки

- название, статус и формат;
- создатель: имя, actor-тип, аватар / логотип, проверка;
- дата, время и место;
- уровень и правила допуска;
- цена, политика возврата и свободные места;
- описание;
- участники;
- правила игрового формата;
- изменения и объявления;
- чат;
- главное контекстное действие.

## Ротация пяти игроков

Дополнительно:

- количество кортов;
- вместимость = корты × 5;
- число раундов;
- модель начисления очков;
- переходы между кортами;
- после публикации — вкладка `Раунды` с личным кортом, партнёром, соперниками и отдыхом.

## Вариант постороннего

Главная кнопка:

- `Присоединиться`;
- `Подать заявку`;
- `В лист ожидания`;
- `Регистрация закрыта`.

До авторизации публичный preview показывает основные сведения, а действие ведёт на вход с возвратом в игру.

## Вариант участника

Показывает:

- статус участия;
- оплату;
- изменения времени / места;
- личный раунд;
- чат;
- результат.

Главная кнопка по состоянию: Оплатить, Подтвердить изменение, Открыть расписание, Открыть чат, Результаты.

Отмена участия находится в меню, но статус и срок отмены видимы рядом с главным действием.

## Вариант организатора

Плашка `Вы организатор` и actor-профиль. Главная кнопка `Управлять`. Рядом показывается ближайшая проблема:

- заявки;
- свободные места;
- неоплаты;
- раунды не опубликованы;
- результаты не заполнены.

## Вкладки

До публикации раундов:

- Обзор;
- Участники;
- Чат.

После публикации:

- Обзор;
- Раунды;
- Участники;
- Чат.

После завершения добавляется `Результаты`.

## Состояния

- loading;
- удалена / отменена;
- перенесена;
- участие ожидает решения;
- платёж истёк;
- пользователь удалён организатором;
- organizer permission lost;
- offline read-only.
