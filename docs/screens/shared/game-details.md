# Детали игры

- Screen ID: `game.details`
- Route: `/games/:gameId`

Канонический экран одноразовой игры.

## Блоки

- название и формат;
- дата, время и место;
- организатор;
- цена и свободные места;
- участники;
- правила;
- чат;
- главное действие.

Для ротации пяти игроков дополнительно показываются количество кортов, вместимость и число раундов.
