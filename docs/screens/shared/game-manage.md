# Управление игрой

- Screen ID: `game.manage`
- Route: `/games/:gameId/manage`

## Разделы

Обзор, Участники, Заявки, Корты и раунды, Результаты, Оплаты, Чат, Уведомления, Настройки, Журнал изменений.

Организатор может менять количество кортов до публикации раундов, но уже опубликованные раунды не пересчитываются самопроизвольно.
