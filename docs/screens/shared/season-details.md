# Детали сезона

- Screen ID: `season.details`
- Route: `/seasons/:seasonId`
- Variants: `guest`, `participant`, `organizer`

## Шапка

- название и статус;
- организатор;
- период сезона;
- формат таблицы;
- правила состава;
- цена / абонемент;
- контекстная кнопка.

## Вкладки

- Обзор;
- Игровые дни;
- Таблица;
- Участники;
- Правила;
- Чат.

## Игровые дни

Карточка дня показывает:

- дату и место;
- регистрацию;
- число кортов;
- вместимость;
- формат игрового дня;
- статус пользователя;
- результаты.

Нажатие ведёт на канонический экран `season.game_day`.

## Таблица

Показывает основной показатель и тай-брейки. Возможны модели:

- командная таблица матчей;
- личные очки игровых дней;
- гибрид.

Любое число открывает расшифровку по исходным игровым дням и матчам.

## Участник

Видит:

- своё место;
- следующий игровой день;
- регистрацию на конкретный день;
- оплату;
- пропущенные дни;
- личные результаты;
- чат.

## Организатор

Плашка `Вы организатор`, кнопка `Управлять`, предупреждения по будущим игровым дням, составу, результатам и оплатам.

## Ротация пяти игроков

Каждый игровой день отдельно хранит:

- `court_count`;
- подтверждённых участников;
- группы по пять;
- раунды;
- результаты;
- движение между кортами.

Изменение количества кортов в одном дне не меняет остальные дни.

## Состояния

- регистрация сезона;
- между игровыми днями;
- регистрация на день;
- игровой день идёт;
- таблица пересчитывается;
- сезон завершён, но не финализирован;
- отменён.
