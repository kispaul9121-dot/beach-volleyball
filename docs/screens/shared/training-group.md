# Тренировочная группа

- Screen ID: `training.group`
- Route: `/training-groups/:groupId`

## Для участника

- название и тренер;
- расписание;
- следующие занятия;
- абонемент и остаток;
- посещаемость;
- материалы;
- прогресс;
- чат группы.

## Для тренера / организации

Вкладки:

- Обзор;
- Участники;
- Расписание;
- Абонементы;
- Посещаемость;
- Прогресс;
- Чат;
- Настройки.

Действия:

- добавить / пригласить участника;
- создать следующее занятие;
- изменить расписание;
- выдать абонемент;
- отметить оплату;
- написать группе;
- архивировать группу.

## Правила

- группа не является отдельной тренировкой; она создаёт и связывает занятия;
- удаление будущего занятия не удаляет группу;
- участник может состоять в группе без записи на каждое занятие, если политика это допускает;
- прогресс и тренерские заметки имеют отдельные настройки приватности;
- архивирование сохраняет историю посещаемости и платежей.
