# Управление тренировкой

- Screen ID: `training.manage`
- Route: `/trainings/:trainingId/manage`

## Дашборд

- записано X из Y;
- заявки;
- неоплаты;
- изменения расписания;
- посещаемость;
- сообщения;
- связанная группа.

## Разделы

- Обзор;
- Программа;
- Участники и заявки;
- Посещаемость;
- Оплаты / абонементы;
- Чат и уведомления;
- Материалы;
- Настройки;
- Аудит.

## Участники

Действия: одобрить, отклонить, перенести из листа ожидания, отметить пробное занятие, применить абонемент, связаться.

## Посещаемость

Для каждого участника: присутствовал, опоздал, уважительный пропуск, неявка. Можно добавить заметку, видимую только тренеру / организации.

После сохранения посещаемость обновляет статистику группы, тренера и участника.

## Программа

Структура занятия, цели, упражнения, оборудование. Участникам показывается только разрешённая публичная часть.

## Изменения

Перенос времени, места или тренера требует уведомления. Критичное изменение может требовать подтверждения участника и разрешать возврат.

## Завершение

Занятие можно завершить после времени окончания. Система напоминает заполнить посещаемость и результаты программы, но финансовые операции остаются отдельными.
